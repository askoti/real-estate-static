**ASKREAL** is a high-end, architectural real estate platform designed for the modern era. It focuses on minimalist aesthetics, glassmorphism, and a "gallery-style" user experience to showcase premium properties globally.

---

## 🏛 The Vision
Unlike traditional real estate platforms, ASKREAL treats properties as pieces of art. The interface is designed with:
- **Architectural Precision**: Generous white space and clean grids.
- **Midnight & Gallery Themes**: A seamless transition between dark hero sections and light-filled property galleries.
- **Dynamic Interaction**: Hover-driven data reveals and smooth transitions.

## 🚀 Key Features
- **Dynamic Property Routing**: Individual, high-fidelity pages for every listing using Next.js Dynamic Routes.
- **Global Concierge**: Integrated Support and Agent Collective pages.
- **Responsive Architecture**: Fully optimized for mobile, tablet, and desktop viewports.
- **Glassmorphic UI**: High-end navigation and search components with backdrop-blur effects.

## 🛠 Tech Stack
- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Typography**: Playfair Display (Serif) & Inter (Sans-serif)
